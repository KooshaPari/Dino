Mod Platform

Not a single mod — a full operating system for DINO modding. 17 projects: Runtime, SDK, Bridge, 4 domain plugins, CLI, MCP server, and more.

📦

Pack System

YAML-first declarative mod packs with explicit dependencies, version ranges, conflict detection, and atomic install/uninstall. 6 example packs included.

⚔️

Four Domain Plugins

Warfare (factions, doctrines, combat), Economy (rates, trade), Scenario (scripting, conditions), and UI (HUD injection, menus).

✅

Schema Validation

17 JSON/YAML schemas validate packs, dependency graphs, asset references, and ECS registration conflicts. 342 tests passing.

🧩

ECS Bridge

Maps cleanly to DINO's full Unity ECS (DOTS) architecture. ECS-native modding over Harmony patches for performance.

🤖

Agent Tooling

MCP server with 13 game interaction tools for Claude Code. CLI with 11 commands. Hot reload. Full agent-driven development pipeline.
