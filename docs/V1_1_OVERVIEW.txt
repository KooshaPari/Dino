================================================================================
                            FIVE PILLARS OF v1.1
================================================================================

1. BUILDING FBX COMPLETION (20-30 hours)
   - Export remaining 20 buildings from Kenney.nl (Tier 1-3)
   - Create 48 faction variants (24 buildings × 2 factions)
   - Target: All buildings < 400 triangles, production-quality
   - Owner: Asset Generation Agent
   - Milestone: March 26 (Tier 1), April 16 (All complete)

2. ADDRESSABLES INTEGRATION (30-40 hours)
   - Implement Unity Addressables for asset streaming
   - Bundle buildings by faction, VFX by type
   - Lazy-load on-demand with memory pooling
   - Target: Load time < 2 sec, memory < 500 MB
   - Owner: SDK/Integration Agent
   - Milestone: March 29 (API complete), April 19 (Production)

3. VFX SYSTEM (40-60 hours)
   - Projectile effects: Blaster, laser, grenade (3 factions)
   - Impact effects: Melee, explosion, environmental
   - Ability activation: Doctrine auras, buff visuals, building effects
   - Environmental: Weather, lighting, audio sync
   - Target: 60 FPS with full VFX, all factions visually distinct
   - Owner: VFX/Graphics Agent
   - Milestone: April 9 (Projectiles + impacts), April 16 (Complete)

4. FACTION UI THEMING (20-30 hours)
   - HUD color customization per faction (health, resources, selection)
   - Menu system theming (main, pause, settings, tooltips)
   - Unit portrait generation with faction-specific styles
   - Loading screens, cinematics, accessibility (colorblind mode)
   - Target: All 3 factions visually & mechanically distinct
   - Owner: UI/Design Agent
   - Milestone: April 19 (Complete)

5. TESTING & POLISH (20-30 hours)
   - Schema validation, reference integrity, completeness checks
   - Balance audit via simulation (win-rate 45-55% target)
   - Integration testing on real DINO instance
   - Performance profiling (memory, CPU, load time)
   - Bug triage, P0/P1 fixes, documentation
   - Target: 450+ tests, zero P0 bugs, release-ready
   - Owner: QA/Testing Agent
   - Milestone: April 26 (Testing complete), May 3 (Polish complete)

================================================================================
                           RISK MITIGATION
================================================================================

Top 5 Risks:
1. FBX batch takes 40+ hours (2x estimate)
   Mitigation: Parallelize with 2-3 artists, prioritize critical buildings

2. Addressables breaks on Mono runtime
   Mitigation: Early bridge IPC testing (week 2), fallback loader ready

3. VFX perf ceiling (can't hit 60 FPS)
   Mitigation: Budget 500 particles/frame, performance mode toggle

4. Balance broken (win-rate 60:40 or worse)
   Mitigation: Simulation + human playtesting both paths, iterate together

5. UI framework too coupled to vanilla DINO
   Mitigation: Use prefab variants + material overrides before Harmony patches

Timeline buffer: 1-2 weeks allocated for unforeseen delays
