================================================================================
INSTALLATION INSTRUCTIONS FOR END USERS
================================================================================

Minimum Requirements:
  - Diplomacy is Not an Option (latest version)
  - BepInEx 5.4.23+ installed
  - DINOForge runtime plugin (included in main release)

Installation:
  1. Extract warfare-starwars folder to: DINO_ROOT/BepInEx/plugins/DINOForge/packs/
  2. Launch DINO normally
  3. Select "Star Wars: Clone Wars" from mod menu
  4. Start a new game with Republic or CIS faction

Verification:
  - Game should display Clone Wars unit models
  - Faction names: "Galactic Republic" vs "Confederacy of Independent Systems"
  - Unit rosters should match those defined in unit YAML files

Troubleshooting:
  - If textures missing: Verify assets/textures/ folder is present
  - If units not loading: Check FBX files in assets/meshes/
  - For errors: Enable debug logging in BepInEx config
